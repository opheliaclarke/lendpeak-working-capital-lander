🔧 LendPeak — Conversion Tracking Spec

Exactly what to build so Google can optimize toward real funded deals. Three owners: WEB landing-page team · GHL GoHighLevel team · ADS us (account side). Account CID 720-674-6191, timezone US Eastern.

The flow in one line: ad click carries a gclid → the page saves it on the lead in GHL → form submit fires a "Lead" conversion → when a deal funds, GHL sends the gclid + amount to a Google Sheet → Google Ads imports it and learns which clicks make money.

Part 1 — Landing pages WEB

1.1 — Install the Google tag on all /lp/* pages

Place the Google Ads global tag (gtag.js, AW-…) + GA4 in the <head> of every /lp/ page. ADS will give you the exact AW-XXXXXXXXX ID and the Lead conversion label — you just install the tag and call the event in step 1.3.

1.2 — Capture the gclid (hidden field + snippet)

Add these hidden inputs to the pre-qualify form, and this snippet to every /lp/ page. It grabs the gclid (+ iOS variants + UTMs) from the URL, fills the fields, and remembers them across multi-step forms.

<!-- hidden fields inside the pre-qualify <form> -->
<input type="hidden" name="gclid"        id="gclid">
<input type="hidden" name="gbraid"       id="gbraid">
<input type="hidden" name="wbraid"       id="wbraid">
<input type="hidden" name="utm_source"   id="utm_source">
<input type="hidden" name="utm_campaign" id="utm_campaign">

<script>
(function(){
  var keys=['gclid','gbraid','wbraid','utm_source','utm_campaign'];
  var p=new URLSearchParams(location.search);
  keys.forEach(function(k){
    var v=p.get(k);
    if(v){ try{ localStorage.setItem('lp_'+k, v); }catch(e){} }
    var el=document.getElementById(k);
    if(el){ try{ el.value = v || localStorage.getItem('lp_'+k) || ''; }catch(e){ el.value = v||''; } }
  });
})();
</script>

1.3 — Fire the "Lead" conversion on submit (with Enhanced Conversions)

On a successful form submit, call this. Enhanced Conversions just means passing the email/phone the user typed — Google hashes it automatically. ADS provides the send_to value.

<script>
// call lpReportLead(email, phone) right after the lead is saved
function lpReportLead(email, phone){
  if(typeof gtag!=='function') return;
  gtag('set','user_data',{ "email": email||"", "phone_number": phone||"" }); // Enhanced Conversions
  gtag('event','conversion',{
    'send_to': 'AW-XXXXXXXXX/XXXXXXXXXXXXXXXX',   // <-- ADS will give you this
    'value': 1.0,
    'currency': 'USD'
  });
}
</script>

Part 2 — GoHighLevel GHL

2.1 Create contact Custom Fields (text): gclid, gbraid, wbraid, utm_source, utm_campaign.
2.2 Map the form's hidden fields → those custom fields, so gclid saves on every contact that comes in.
2.3 Confirm a pipeline stage (or opportunity status) that means Funded — e.g. a "Funded" stage.
2.4 Build one Workflow: Trigger = opportunity moves to the Funded stage. Action = Add row to Google Sheet (Google Sheets action, or Webhook/Zapier) with the columns below. Use the contact's saved gclid, and the funded date/time in Eastern time.

Google Sheet columns — must be exactly these (Google's offline-import format):

Google Click ID     -> {{contact.gclid}}
Conversion Name     -> Funded Deal
Conversion Time     -> funded date/time, e.g. 2026-06-26 15:30:00 America/New_York
Conversion Value    -> funded amount, e.g. 25000
Conversion Currency -> USD
If a lead has no gclid (e.g. organic), just leave that row out — only push deals that came from a click with a gclid.

Part 3 — Google Ads ADS (our side, no action for Leo's team)

  • Create conversion action "Lead" (form submit) with Enhanced Conversions ON → hand WEB the send_to.
  • Create conversion action "Funded Deal" (import, value-based).
  • Tools → Conversions → Import → Google Sheets → connect the Sheet, daily auto-import.
  • Bid to Lead at launch; switch to Funded Deal once that data builds.

Scheduling ADS

Team hours: 9am–5pm PT = 12pm–8pm ET. Account is on Eastern, so the ad schedule goes in as Mon–Fri, 12:00 PM – 8:00 PM ET. This keeps spend in the window where leads get a same-hour callback. (Confirm whether weekends are staffed — if not, we leave them off.)
